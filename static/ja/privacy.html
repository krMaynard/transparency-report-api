プライバシーポリシー

最終更新：2026 年 6 月 · お問い合わせ：kieranmaynard@gmail.com

要約：本サービスは、API アクセスの提供と不正利用の防止に必要な最小限の情報のみを収集します。プロンプトは Anthropic の Claude API に転送されますが、ここには保存されません。分析ツール、広告、第三者によるトラッキングは一切ありません。

本サービスについて

Transparency Report API は独立した研究ツールであり、構造化された透明性データ（EU デジタルサービス法（DSA）のコンテンツモデレーション統計と Google への政府削除要請）を、構造化された HTTP API と対話型ダッシュボードを通じてクエリ可能にします。欧州委員会、Google、およびそのデータを扱う対象プラットフォームのいずれとも提携していません。

サインインまたは登録が求められる理由

構造化クエリ API（POST /api/query、GET /api/jobs/*）へのアクセスには API キーが必要です。理由は 2 つあります：

  • レート制限。サーバーリソースの不正利用を防ぐため、各キーは 60 秒あたり 60 クエリに制限されています。
  • ジョブの分離。クエリのジョブは、それを作成したキーごとに分離されます — 他人の結果は見えず、自分の結果のみが表示されます。

広告、データの収益化、アカウント情報の商業利用は一切ありません。

アクセスを取得する方法

2 つの方法があります：

  • Google サインイン。Google アカウントでサインインすると、セッションキーが自動的に作成されます。Google から提供されるメールアドレスはアカウント識別子として保存され、レート制限の適用と、必要に応じたアクセスの取り消しのためにのみ使用されます。Google のパスワードが本サービスに見られることはありません — 認証はすべて Google Identity Services が処理します。
  • デモ登録（有効な場合）。ポータルでメールアドレスを送信すると、短期間のデモキーが発行されます。メールアドレスはそのキーの識別子として保存されます。

上記に記載されている以外の氏名、支払い情報、その他の個人データは一切収集しません。

「質問」機能 — Anthropic に送信されるプロンプト

自然言語による質問ボックス（POST /api/ask）では、クエリを自然な言葉で記述できます。プロンプトを送信すると：

  • 入力した自由記述のプロンプトは、構造化クエリに変換するためにリアルタイムで Anthropic の Claude API に転送されます。
  • 本サービスはプロンプトを保存しません — プロンプトは通過するだけで、ここでディスクやログに書き込まれることはありません。
  • Anthropic は、独自のプライバシーポリシーに従ってプロンプトを記録・処理する場合があります。
  • この機能を使用したくない場合は、対話型クエリビルダーまたは API を直接利用してください — どちらも第三者の LLM 呼び出しなしで動作します。

この機能は、デプロイ環境に ANTHROPIC_API_KEY が設定されている場合にのみ有効です。

Google Identity Services サインイントークンの検証 policies.google.com/privacy

本サイトは、Google Identity Services がサインイン状態のために設定するもの以外に、分析ツール、広告ネットワーク、トラッキングピクセル、Cookie を使用しません。

データの削除とお問い合わせ

API キーとセッションは短命です。それぞれ有効期間（TTL）が設定され、メモリ上にのみ保持されるため、期限切れ時またはサーバー再起動時に自動的に破棄されます。クエリ結果がディスクに書き込まれることはありません。

その他のご質問やデータ削除のご依頼は、こちらまでお問い合わせください：kieranmaynard@gmail.com
