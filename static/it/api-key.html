Richiedi l’accesso all’API

Accedi per ottenere una chiave API.

Una chiave API sblocca l’API di interrogazione asincrona —invia un job, esegui il polling ed esporta i risultati completi in JSON/CSV— oltre al riquadro Ask in linguaggio naturale, alle callback webhook e a limiti di utilizzo più elevati. Per esplorare lo schema ed eseguire le interrogazioni limitate dei pannelli non serve alcuna chiave.

Accedi con un qualsiasi account Google: la tua chiave API viene emessa immediatamente.

oppure ottieni una chiave demo

Senza verifica: questo emette una chiave demo temporanea per provare l’API. Usa un nome e un’email qualsiasi.
