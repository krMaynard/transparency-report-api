이 서비스란

Transparency Report API는 구조화된 투명성 데이터 — EU 디지털 서비스법(DSA) 콘텐츠 조정 통계와 Google 정부 삭제 요청 — 를 구조화된 HTTP API와 대화형 대시보드를 통해 쿼리할 수 있게 하는 독립적인 연구 도구입니다. 유럽 위원회, Google, 또는 데이터가 다뤄지는 어떤 플랫폼과도 제휴 관계가 없습니다.

로그인 또는 등록을 요청하는 이유

구조화 쿼리 API(POST /api/query, GET /api/jobs/*)에 접근하려면 API 키가 필요합니다. 두 가지 이유가 있습니다:

  • 요청 빈도 제한. 서버 리소스의 악용을 방지하기 위해 각 키는 60초 창당 60건의 쿼리로 제한됩니다.
  • 작업 격리. 쿼리 작업은 이를 생성한 키로 범위가 한정됩니다 — 다른 사람의 결과가 아닌 본인의 결과만 볼 수 있습니다.

광고, 데이터 수익화, 계정 정보의 상업적 이용이 없습니다.

접근 방법

두 가지 옵션이 있습니다:

  • Google 로그인. Google 계정으로 로그인하면 세션 키가 자동으로 생성됩니다. Google이 제공한 이메일 주소는 계정 식별자로 저장되며, 요청 빈도 제한을 적용하고 필요 시 접근을 취소할 수 있도록 하는 데에만 사용됩니다. 이 서비스는 Google 비밀번호를 절대 볼 수 없으며 — 인증은 전적으로 Google Identity Services가 처리합니다.
  • 데모 등록(활성화된 경우). API 키 페이지에 이메일 주소를 제출하면 단기 데모 키가 발급됩니다. 이메일은 해당 키의 식별자로 저장됩니다.

수집되는 데이터

항목목적보존 기간
Google 계정 이메일 (Google로 로그인하는 경우) 계정 식별, 요청 빈도 제한, 관리자 취소 세션이 만료될 때까지(또는 서버가 재시작될 때)
데모 등록 시 제공한 이메일 키 식별자, 요청 빈도 제한 데모 키가 만료될 때까지 (일반적으로 24시간)
API 키 (불투명 토큰) 인증, 요청 빈도 제한 TTL로 제한됨; 서버 재시작 시 삭제됨 (Redis가 구성되지 않은 한 메모리 내 저장)
쿼리 결과 작업 폴링 API를 통해 사용자에게 반환됨 메모리 내에만 저장; 디스크에 절대 기록되지 않음; 서버 재시작 시 삭제됨

위에 나열된 것 외에는 이름, 결제 정보 또는 기타 개인 데이터를 수집하지 않습니다.

"질문" 기능 — Anthropic으로 전송되는 프롬프트

자연어 질문 상자(POST /api/ask)를 사용하면 쿼리를 자연어로 설명할 수 있습니다. 프롬프트를 제출하면:

  • 자유 텍스트 프롬프트는 구조화된 쿼리로 변환되도록 실시간으로 Anthropic's Claude API에 전달됩니다.
  • 이 서비스는 프롬프트를 저장하지 않습니다 — 통과될 뿐 여기의 디스크나 로그에 기록되지 않습니다.
  • Anthropic은 자체 개인정보 처리방침에 따라 프롬프트를 기록하고 처리할 수 있습니다.
  • 이 기능을 사용하지 않으려면 대화형 쿼리 빌더나 API를 직접 사용하세요 — 둘 다 제3자 LLM 호출 없이 작동합니다.

이 기능은 배포 환경에 ANTHROPIC_API_KEY가 구성된 경우에만 활성화됩니다.

이 사이트는 Google Identity Services가 로그인 상태를 위해 설정하는 것 외에는 분석 도구, 광고 네트워크, 추적 픽셀 또는 쿠키를 사용하지 않습니다.

데이터 삭제 및 문의

API 키와 세션은 수명이 짧습니다. 각각 수명(TTL)이 정해져 있고 메모리에만 보관되므로 만료되거나 서버가 재시작될 때 자동으로 폐기됩니다. 쿼리 결과는 절대 디스크에 기록되지 않습니다.
