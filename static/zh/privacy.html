隐私政策

最后更新：2026 年 6 月 · 疑问：kieranmaynard@gmail.com

简短版本：本服务仅收集为你提供 API 访问权限和防止滥用所需的最少信息。你的提示词会转发至 Anthropic 的 Claude API，但不会在此存储。没有分析统计、没有广告、没有第三方跟踪。

本服务是什么

Transparency Report API 是一款独立的研究工具，通过结构化的 HTTP API 和交互式仪表盘，让结构化透明度数据——欧盟《数字服务法》（DSA）内容审核统计数据和 Google 政府删除请求——变得可查询。它与欧盟委员会、Google 或其所涵盖数据的任何平台均无关联。

为什么要求你登录或注册

访问结构化查询 API（POST /api/query、GET /api/jobs/*）需要 API 密钥。原因有两个：

  • 速率限制。每个密钥每 60 秒窗口内最多允许 60 次查询，以防止滥用服务器资源。
  • 任务隔离。查询任务仅限于创建它们的密钥——你只能看到自己的结果，看不到他人的。

没有广告，不会将数据货币化，也不会对你的账号信息进行任何商业利用。

如何获取访问权限

有两种方式可供选择：

  • Google 登录。使用 Google 账号登录会自动创建一个会话密钥。你由 Google 提供的电子邮件地址会作为账号标识符存储，仅用于实施速率限制以及在必要时撤销访问权限。本服务绝不会看到你的 Google 密码——身份验证完全由 Google Identity Services 处理。
  • 演示注册（在启用时）。在 API 密钥页面提交电子邮件地址会签发一个短期有效的演示密钥。该电子邮件会作为该密钥的标识符存储。

所收集的数据

内容原因保留期
Google 账号电子邮件（如果你使用 Google 登录） 账号身份、速率限制、管理员撤销 直至你的会话过期（或服务器重启）

除上述所列内容外，不会收集姓名、支付信息或其他个人数据。

“提问”功能——发送至 Anthropic 的提示词

自然语言提问框（POST /api/ask）让你能够用自然语言描述查询。当你提交提示词时：

  • 你的自由文本提示词会实时转发至 Anthropic 的 Claude API，以便翻译为结构化查询。
  • 本服务不会存储你的提示词——它只是经过本服务，不会在此写入磁盘或日志。
  • Anthropic 可能会依据其自身的隐私政策记录和处理提示词。
  • 如果你不想使用此功能，可直接使用交互式查询构建器或 API——两者均无需任何第三方 LLM 调用即可工作。

仅当在部署环境中配置了 ANTHROPIC_API_KEY 时，此功能才会启用。

数据删除与联系方式

API 密钥和会话都是短时的：每个都有生存时间（TTL）限制，且仅保存在内存中，因此会在过期或服务器重启时自动丢弃。查询结果绝不会写入磁盘。
